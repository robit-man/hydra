## Success Metrics

- Each view has uniform title, divider, and footer behavior.

- Hydra/market metadata remains visible without overlapping animated art.

## Failure Modes + Preventive Controls

- Failure: Content density overwhelms smaller terminals.
  - Prevention: per-view compact mode and prioritized field elision.
- Failure: Text truncation hides critical state (owner key hint, sync state).
  - Prevention: reserve fixed slots for high-priority metadata.
- Failure: Animation redraw damages static panel text.
  - Prevention: strict layering order and explicit dirty-region redraw.
