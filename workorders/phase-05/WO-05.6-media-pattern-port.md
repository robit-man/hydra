## Success Metrics
- Media stream continues through fallback when tunnel dies.
- UI indicates active transport (`cloudflare`, `nkn`, etc.) for media path.
- No hard crash on stale tunnel or temporary stream errors.

## Potential Failure Modes And Preventions
- Failure mode: excessive media retries saturate NKN.
  Prevention: bounded retry policy and adaptive frame rate downgrade.
- Failure mode: stale media URLs cached indefinitely.
  Prevention: clear cache on stale/error state and re-resolve.
- Failure mode: audio/video desync in fallback.
  Prevention: independent jitter buffers and timestamp tracking.
