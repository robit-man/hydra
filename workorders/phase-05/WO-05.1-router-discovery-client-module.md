## Potential Failure Modes And Preventions
- Failure mode: multiple concurrent resolve requests race.
  Prevention: single in-flight resolve guard and stale-response discard.
- Failure mode: bad router address breaks all updates.
  Prevention: address validation + non-destructive failure state.
- Failure mode: auto-resolve floods network.
  Prevention: minimum interval and jitter/backoff on repeated failures.
