## Success Metrics
- Logs show state transitions: `launching`, `activating`, `running`, `degraded`, `error`, `stopping`.
- Services with temporary probe failure stay `degraded` before hard restart.
- State transitions are visible in router status output.

## Potential Failure Modes And Preventions
- Failure mode: aggressive restart loops.
  Prevention: exponential backoff and activation timeout windows.
- Failure mode: false negative probes for auth-protected endpoints.
  Prevention: treat `401/403` as process-up for activation checks.
- Failure mode: stale state after crash.
  Prevention: reset runtime counters/flags on relaunch paths.

## Verification

- Force probe failures and confirm degraded-before-error behavior.
